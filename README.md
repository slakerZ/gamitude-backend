# gamitude-backend
## backend + database
docker-compose build  
after repository update run docker-compose build --no-cache  
docker-compose run (-d)  
## database   
```
cd db  
docker-compose up
```
## server  
```
cd gamitude_backend  
dotnet run  
```
default starts at localhost http 5000 / https 5001  
## swagger ui
api/swagger endpoint
